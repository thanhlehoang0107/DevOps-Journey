# Exercises - Linux & Bash

## Bài tập 1: Quản lý Files

Tạo cấu trúc thư mục sau và các files tương ứng:

```
project/
├── src/
│   ├── main.sh
│   └── utils.sh
├── docs/
│   └── README.md
└── tests/
    └── test.sh
```

## Bài tập 2: Permissions

Cho file `deploy.sh`, thực hiện:

1. Cho phép owner thực thi
2. Cho phép group đọc
3. Không cho phép others truy cập

## Bài tập 3: Bash Script

Viết script `backup.sh` thực hiện:

1. Tạo thư mục backup với timestamp
2. Copy tất cả files .txt vào thư mục đó
3. Hiển thị thông báo hoàn thành
