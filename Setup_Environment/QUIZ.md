# ❓ Quiz - Setup Environment

> **Knowledge Check for Environment Setup**
>
> *Kiểm tra kiến thức cài đặt môi trường*

## 📋 Questions (Câu hỏi)

### Q1: What is WSL2? (WSL2 là gì?)

- A) Windows Software Library
- B) Windows Subsystem for Linux ✓
- C) Web Server Linux
- D) Windows System Loader

---

### Q2: Which command checks Docker version? (Lệnh nào kiểm tra phiên bản Docker?)

- A) `docker version`
- B) `docker --version`
- C) `docker -v`
- D) All of the above (Tất cả đều đúng) ✓

---

### Q3: What is the default package manager on macOS for DevOps? (Package manager mặc định trên macOS?)

- A) apt
- B) yum
- C) Homebrew ✓
- D) npm

---

### Q4: Which tool is used to manage Kubernetes? (Tool nào quản lý Kubernetes?)

- A) docker
- B) kubectl ✓
- C) terraform
- D) ansible

---

### Q5: What is Trivy used for? (Trivy dùng để làm gì?)

- A) Build Docker images
- B) Scan container vulnerabilities ✓
- C) Deploy applications
- D) Monitor systems

---

### Q6: Which command tests SSH connection to GitLab? (Lệnh nào kiểm tra kết nối SSH với GitLab?)

- A) `ssh git@gitlab.com`
- B) `ssh -T git@gitlab.com` ✓
- C) `git connect gitlab`
- D) `ping gitlab.com`

---

### Q7: What is the primary CI/CD platform in this course? (Nền tảng CI/CD chính trong khóa học này?)

- A) Jenkins
- B) GitHub Actions
- C) GitLab CI ✓
- D) CircleCI

---

## ✅ Answers (Đáp án)

| Q | Answer | Explanation (Giải thích) |
|---|--------|--------------------------|
| 1 | B | WSL allows running Linux on Windows |
| 2 | D | All commands show Docker version |
| 3 | C | Homebrew is the popular package manager for macOS |
| 4 | B | kubectl is the CLI for Kubernetes |
| 5 | B | Trivy scans for security vulnerabilities |
| 6 | B | `-T` flag tests authentication without command |
| 7 | C | GitLab CI is the primary platform |
